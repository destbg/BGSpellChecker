Notepad Help

  • The translator only works in Bulgarian, although i did write everything in the site in English
  • If the previous session's text is missing:
    • Enable browser cookies and history.
    • Do not use browser's private/incognito mode. The text local storage will be deleted by the browser when you close the browser's window.
  • In order to correct a word click on the red underline and wait for the list of correct words to appear
  • The circle on the bottom right will display the number of incorrect words
  • The saved notepad's text will be deleted when you delete your browsing history/cache or run disk cleaning application (E.g. Windows Disk cleanup / CCleaner).
  • The notepad's text is auto saved on page close and on tab blur.
  • The notepad's text will not be saved with private mode browsing.
  • To open saved file, look for the file in the Downloads folder.
  • Save button saves the file to the Downloads folder. See: Where do files go when downloaded?
